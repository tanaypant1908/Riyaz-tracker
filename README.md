# Riyaz Tracker – Indian Classical Music Learning System

Riyaz Tracker is a lightweight, responsive web application designed to help Indian Classical Music students structure their daily practice and master the foundational grammar of Hindustani Ragas. 

This is a complete, serverless frontend application that runs entirely in the web browser, utilizing local storage for persistent data tracking.

## 🌟 Key Features

- **Comprehensive Raga Library:** Pre-loaded with complete melodic profiles for 20 popular ragas (including Yaman, Bhairav, Bhupali, Bageshree, Malkauns, and more).
- **Core Musical Grammar:** Displays the critical components of each raga: Short Introduction, Aaroh (ascending scale), Avroh (descending scale), Pakad (signature phrase), and 3 Sample Taans.
- **Interactive Step-by-Step Learning:** A 6-step checklist for each raga that visually updates individual progress bars and awards a "Prepared" badge upon completion.
- **Smart Practice Logger:** Allows students to log practice dates, select ragas, input practice minutes, and add personal training notes.
- **Dynamic Analytics Dashboard:** Calculates total riyaz hours/minutes, ongoing daily practice streaks, and an overall percentage-based learning progress wheel.
- **Advanced Filtering:** Instant, real-time filtering of ragas by explicit **Mood/Rasa** (Devotional, Calm, Romantic, Serious, Joyful) and **Time of Performance** (Prahar).

## 🛠️ Built With

- **HTML5:** Semantic structure for presentation.
- **CSS3:** Custom responsive layout, Indian classical-inspired color palette, and dynamic progress animations.
- **JavaScript (ES6):** State management, real-time data calculations, search/filter algorithms, and browser DOM manipulation.
- **Web Storage API:** Utilizes `LocalStorage` to securely save student progress and practice logs locally without requiring a database backend.

## 🚀 How to Run the Project Locally

1. Clone or download this repository to your local machine.
2. Ensure `index.html`, `style.css`, and `script.js` are saved in the same directory.
3. Simply double-click `index.html` to launch the application instantly in any modern web browser.
